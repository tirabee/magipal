# Magipal Roadmap

## Core Features

- [x] Sidebar palette list with swatches
- [x] Palette selection + color swatch view
- [x] Click to copy hex value
- [x] Create / rename / delete palettes
- [x] Add / remove individual colors from a palette
- [x] Manual color picker (within app)
- [x] Persist palettes to disk (local JSON file)
- [x] Lock/unlock feature for palettes
- [ ] Add Dither Test to palette view
- [ ] Analagous palette generation

## Organization

- [x] Folders / projects to group palettes
- [ ] Tag palettes with moods/vibes

## Import

- [x] Import palette from PNG swatch grid
- [ ] Import palette from URL (colourlovers etc.)

## Color Tools

- [x] System-wide eyedropper (pick color from anywhere on screen)
- [ ] Harmony suggester (complementary, analogous, triadic)
- [ ] Contrast / WCAG accessibility checker

## Export

- [x] Export as PNG swatch sheet
- [x] Export as CSS variables
- [x] Export as hex list
- [x] Export as JSON
- [x] Export as ASE
- [x] Export as GPL

## Polish

- [ ] Smooth transitions between palette selections
- [ ] Keyboard shortcuts
